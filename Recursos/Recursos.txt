
Api : https://www.themealdb.com/api.php

Texto : Buscador de Recetas

Página de Inicio:

En la página de inicio, podrías mostrar una lista de categorías de comidas disponibles utilizando la URL www.themealdb.com/api/json/v1/1/categories.php.
Cada categoría podría ser un enlace que dirija a una página donde se muestran las recetas de esa categoría específica.
Página de Categoría:

Cuando un usuario selecciona una categoría, puedes utilizar la URL www.themealdb.com/api/json/v1/1/filter.php?c={category} para obtener todas las recetas de esa categoría.
Muestra las recetas obtenidas con imágenes y nombres, y permite a los usuarios hacer clic en una receta para ver más detalles.
Página de Detalles de Receta:

Cuando un usuario hace clic en una receta, puedes utilizar la URL www.themealdb.com/api/json/v1/1/lookup.php?i={id} para obtener los detalles completos de esa receta específica.
Muestra la imagen de la receta, el nombre, los ingredientes y los pasos de preparación.
Barra de Búsqueda:

Agrega una barra de búsqueda en la parte superior de la página que permita a los usuarios buscar recetas por nombre. Utiliza la URL www.themealdb.com/api/json/v1/1/search.php?s={query} para buscar recetas por nombre.
Funcionalidades Adicionales:

Implementa la capacidad de guardar recetas favoritas utilizando almacenamiento local en el navegador.
Agrega filtros por tipo de comida (desayuno, almuerzo, cena) utilizando la URL www.themealdb.com/api/json/v1/1/filter.php?c={category} con la categoría correspondiente.
Considera agregar una opción para mostrar recetas aleatorias utilizando la URL www.themealdb.com/api/json/v1/1/random.php.
